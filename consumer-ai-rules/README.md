# Instrucciones AI para proyectos que consumen flysoft-react-ui

Esta carpeta contiene archivos de instrucciones listos para copiar en tu proyecto.
Cada archivo contiene las mismas reglas, pero nombrado para la herramienta AI correspondiente.

## Uso

Copia **uno** de estos archivos en tu proyecto según la herramienta AI que uses:

| Herramienta | Archivo a copiar | Destino en tu proyecto |
|-------------|-----------------|----------------------|
| **Claude Code** | `CLAUDE.md` | Raíz del proyecto (`./CLAUDE.md`) |
| **Cursor AI** | `.cursorrules` | Raíz del proyecto (`./.cursorrules`) |
| **GitHub Copilot** | `copilot-instructions.md` | `.github/copilot-instructions.md` |
| **Windsurf (Codeium)** | `.windsurfrules` | Raíz del proyecto (`./.windsurfrules`) |
| **Otro / generico** | `flysoft-react-ui-rules.md` | Donde tu herramienta AI lea instrucciones |

## Notas

- El contenido es identico en todos los archivos, solo cambia el nombre.
- Si usas multiples herramientas AI, copia el archivo correspondiente a cada una.
- Estos archivos se actualizan con cada version de la libreria. Recuerda actualizarlos cuando actualices `flysoft-react-ui`.
